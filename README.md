## Q2: 實作 Middleware

### Get Started
#### 0. Prerequisites/Environment
* Node.js v10.15.0
* express v4.17.1
* nodemon: v1.19.1
    
#### 1. Clone this app
```
$ git clone https://github.com/jacs0110/core_final_exam_q2.git
```

#### 2. Install npm packages
```
$ npm install
```

#### 3. Run the server
```
$ npm run dev
```

and you will see "App running on port 3000" in the console
